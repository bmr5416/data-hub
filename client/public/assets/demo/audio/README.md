# Demo Video Background Music

Place a background music file here for the demo video.

## Recommended Track

**"Fantasy Medieval Ambient"** by DeusLower on Pixabay

- **Duration:** 3:39 (perfect for ~3 minute demo video)
- **License:** Pixabay Content License (free for use, no attribution required)
- **Style:** Medieval, Fantasy, Ambient, Folk, Tavern
- **URL:** https://pixabay.com/music/folk-fantasy-medieval-ambient-237371/

## How to Download

1. Visit the Pixabay URL above
2. Click "Download" (requires free Pixabay account)
3. Download the MP3 file
4. Rename to `background.mp3` and place in this directory

## File Requirements

- **Filename:** `background.mp3`
- **Format:** MP3
- **Duration:** Should be at least as long as the demo video (~3 minutes)

## Alternative Tracks

If the recommended track isn't available, search for:
- "medieval ambient music" on Pixabay
- "fantasy tavern music" on Pixabay
- "dungeon ambient" on OpenGameArt

## Usage

The `compile-demo.js` script will automatically:
- Fade in the music over 2 seconds at the start
- Fade out the music over 3 seconds at the end
- Mix at an appropriate volume level

If no music file is found, the demo video will be silent (which is also acceptable).
